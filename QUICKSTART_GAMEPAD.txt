1. Make sure MQTT broker is running:
   sudo systemctl start mosquitto

2. If using real lights, pair them:
   - Open http://localhost:8080
   - Click "Permit Join"
   - Reset bulbs (on/off 5x rapidly)

   sudo usermod -a -G input $USER
   (then log out/in)

4. Launch:
   bash launch_gamepad.sh
